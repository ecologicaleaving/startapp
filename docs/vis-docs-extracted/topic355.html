VolleyLiveMatchIdentificationAnswer Schema
Description

Answer from a client identification for a volleyball match data.

Elements
ElementDescription
AcceptedThe identification is accepted.
BadMatchNoIdentification refused: the match number is not correct.
BadUsernameOrPasswordIdentification refused: bad username or password.
InvalidMatchNoIdentification refused: invalid match number.
InvalidRoleIdentification refused: invalid role value.
InvalidVersionIdentification refused: invalid version.
VersionNotSupportedIdentification refused: cannot communicate with the requested version.
