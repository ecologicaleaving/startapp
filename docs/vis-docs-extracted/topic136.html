Volleyball > Transfers > Volley Transfer Type
Volley Transfer Type

Type of a volleyball transfer.

 Values

The following table lists the values for the volleyball transfer types. 

Value

 Name Description

1

 International Transfer to a club which is in a division which directly or through promotion has access to international tournaments or matches.

2

 University When the receiving club is a university club which is in a division which directly or through promotion has access to international tournaments or matches.

3

 NoIntlRights Transfer to a club in a division which neither directly nor through promotion, has access to international tournaments or matches.
