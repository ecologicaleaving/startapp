Volleyball Tournament Ranking Team Source

The team source gives an indication of which team must be at a specified rank for a tournament ranking.

It can take one of the following form:

  • Loser of a match.
  • Specified rank in a round.
  • Winner of a match.
 Loser of a match

The team at the specified rank is the loser of a match.

The XML representation is the following:

Loser of a match
Copy Code 
<Loser NoMatch="<NoMatchInTournament>" />

Where NoMatchInTournament is the number of the match in the tournament.

 Specified rank in a round

The team at the specified rank is the team ranked at a specific rank in a round.

The XML representation is the following:

Specific rank in a round
Copy Code 
<RoundRank Rank="<Rank>" RoundCode="<RoundCode>" />

Where Rank is the rank in the round and RoundCode is the code of the round in the tournament.

 Winner of a match

The team at the specified rank is the winner of a match.

The XML representation is the following:

Winner of a match
Copy Code 
<Winner NoMatch="<NoMatchInTournament>" />

Where NoMatchInTournament is the number of the match in the tournament.
