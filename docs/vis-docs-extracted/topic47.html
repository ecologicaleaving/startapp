Press Release

Represents a press release.

The name of the XML element for a press release text is <PressRelease>.
The name of the XML element for a list of press release texts is <PressReleases>.

 Fields

The following table lists all the fields for a press release.

Name Type Description
Category PressReleaseCategory Press release category.
DateTime DateTime Date and time of the press release.
DeletedDT DateTime Null if the event is valid.
Date and time of the deletion when the event is deleted.
Languages String

ISO 639-1 code of the languages for the text. If the press release is available in more than one language, this field contains the list of the codes of all the languages, separated with spaces.

Currently, the following languages are used in press release texts:

Code Language
de German
en English
es Spanish
fr French

Almost all of the press release have an English version.
Only some of them are translated to other languages.
LastChangeDT DateTime Date and time of the last change in the text's data.
LastChangeUsername String Username of the user who has made the last change in the text's data.
No Int32 Unique number of the press release.
This number is unique among all the press releases.
NoMatch Int32 Number of the match to which the press release is related to.
