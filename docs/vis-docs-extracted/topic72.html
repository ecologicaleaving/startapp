Discrete values > Beach volleyball > Beach Tournament Action
Beach Tournament Action

Available actions for a beach volleyball tournament.
The values are OR-ed to have the final value, which is a 32-bits integer.

 Values

The following table lists the values for the available actions for a beach volleyball tournament.
Note that the Value column contains hexadecimal values.

Value Name Description
0x0001 Administrator An administrator has all accesses.
0x0002 CanRegisterTeam A user can register a new team.
0x0004 CanUnregisterTeam A user can unregister a team.
0x0008 CanDeleteTeam A user can delete a team.
0x0010 CanWithdrawTeam A user can withdraw a team.
0x0020 CanChangePlayer A user can change a player in a team.
0x0040 CanDefineReserveTeam A user can define a reserve team.
