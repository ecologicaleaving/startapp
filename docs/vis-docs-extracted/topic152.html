Volleyball Player Filter

Filter to select players registrations in volleyball tournaments.

 Properties

The following table lists the properties for the filter.

Name Type Description
ForPublicDisplay Boolean

Selects only the registrations for the public display.

If not specified or the value is true, the selected players registrations depend on the value of the PlayerDisplayMethod property of the tournament on which the player is registered, as indicated in the following table:

PlayerDisplayMethod Selected players registrations
All All registrations.
None No registrations.
Preselected Only the preselected players.
Selected Only the selected players.
WithShirtNumber Only the players with shirt numbers.
NoEvent Int32 Selects the players registered in an event.
NoPlayer Int32 Selects the registrations of a player.
NoTeam Int32 Selects the players registered for a team registration.
NoTournament Int32 Selects the players registered in a tournament.
NoTournaments Int32 set Selects the players registered in multiple tournaments.
Numbers Int32 set Selects the players registrations with the specified numbers.
 Examples
Filter Description
<Filter NoPlayer="141429" /> Selects the registrations of the player with number 141429.
<Filter NoTeamr="3004" /> Selects the players registrations for the team registration with number 3004.
<Filter NoTournament="979" /> Selects the players registered in the tournament with the number 979.

<Filter NoTournament="979"
        ForPublicDisplay="True" />

 Selects the players registered in the tournament with the number 979 and that should be displayed publicly.
