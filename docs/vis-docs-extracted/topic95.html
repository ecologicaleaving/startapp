Press Release Publish On

Where a press release must be published.

 Values

The following table lists the values that indicates where a press release must be published.

Value Name Description

1

 Beach Publish on beach volleyball pages.

2

 Development Publish on development pages.

4

 Fivb Publish on FIVB pages.

5

 Headlines Publish on headlines.

6

 Home Publish on home page.

7

 Medical Publish on medical pages.

8

 President Publish on president pages.

9

 Refereeing
RefereeingRules
Rules		 Publish on refereeing and rules pages.

10

 Snow Publish on snow volleyball pages.

11

 Coach
Technical
TechnicalCoach		 Publish on technical and coach pages.

12

 Tournament Publish on tournament pages.

13

 Twitter Publish on Twitter.

14

 Volleyball Publish on volleyball pages.

15

 VolleyballWorld Publish on the Volleyball World publication.

16

 WorldVolleyNews Publish on the World Volley News publication.

17

 Msdp Publish on the Microsoft Sports Digital Portal.


The value 15 was previously assigned to the VolleyWorld value (VolleyWorld publication). As this publication no more exists and has been replaced by the Volleyball World publication, this value has been renamed.

Some values have more than one name: in this case, the web service accepts any of the listed names.
