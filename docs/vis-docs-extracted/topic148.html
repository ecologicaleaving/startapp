Filter for an article

Filter for articles.

 Attributes

The following table lists the attributes for the filter.

Name Type Description
PublishOn PressReleasePublishOn Where the articles should be published.
Each location must be specified by its name or value, and, if more than one location is specified, they must be separated with spaces.
If it is not specified, the returned articles are not filtered by this criteria.


In addition to the previously listed attribute, you can use Tags Filtering.
