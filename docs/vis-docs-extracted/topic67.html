Beach Olympic Team Status

Status for a team in the ranking for the Olympic Games selection.

 Values

The following table lists the values for the statuses for a team in the ranking for the Olympic Games selection.

Value

 Name Description

1

 Selected Team selected.

2

 SelectedMinHostQuota Team selected because of the host quota.
There is a minimum number of teams for the Olympic Games host country.

3

 SelectedMinConfederationQuota Team selected because of the confederation quota.
There is a minimum number of teams for each confederation.

4

 Tie Team not selected, but tied with another one.

5

 NotEnoughTournaments Team not selected because it has not played enough tournaments during the selection period.

6

 CountryQuota Team not selected because of the country quota.
There is a maximum number of teams for each country.

7

 NotRegistered Team not selected because it has not been registered for the Olympic Games tournament.

8

 NotEnoughPoints Team not selected because it has not enough points.
