Volleyball Transfer Permissions

Permissions for a volleyball transfer.

 View permissions

The following table lists the view permissions for a volleyball transfer.

Name Description
CanViewCertificate Gives the user the permission to view the certificate for a volleyball transfer.
This permission can be limited to some transfers for some users.
CanViewDetails Gives the user the permission to view the details of a volleyball transfer.
This permission can be limited to some transfers for some users.
CanViewPayment Gives the user the permission to view the data of a payment for a volleyball transfer as the sender of the money.
This permission can be limited to some transfers for some users.
CanViewPaymentAsFivb Gives the user the permission to view all the data of a payment for a volleyball transfer.
This permission can be limited to some transfers for some users.
CanViewReceipt Gives the user the permission to view the receipt of the payments for a volleyball transfer.
This permission can be limited to some transfers for some users.
 Insert permissions

The following table lists the insert permissions for a volleyball transfer.

Name Description
CanInsert

Gives the user the permission to insert a new volleyball transfer.

The following parameters can be used with this permission.

Name Description
ClubTo When this parameter is specified, it contains the number of a volleyball club.
The transfer will be limited to the club of destination with the specified number.
Federation When this parameter is specified, it contains the code of a federation.
The transfer will be limited to a club of destination to the federation with the specified code.
 Change permissions

The following table lists the change permissions for a volleyball transfer.

Name Description
CanCancel Gives the user the permission to cancel a volleyball transfer.
This permission can be limited to some transfers for some users.
CanConfirm Gives the user the permission to confirm a volleyball transfer.
This permission can be limited to some transfers for some users.
CanDelete Gives the user the permission to delete a volleyball transfer.
This permission can be limited to some transfers for some users.
CanEditPayment Gives the user the permission to edit a payment for a volleyball transfer as the sender of the money.
This permission can be limited to some transfers for some users.
CanEditPaymentAsFivb Gives the user the permission to edit all the data for a payment for a volleyball transfer.
This permission can be limited to some transfers for some users.
CanGenerateCertificateAgain Gives the user the permission to generate a new certificate for a transfer.
This permission doesn't control the automatically generated certificates, when a transfer is completed or when a transfer release is completed.
This permission can be limited to some transfers for some users.
CanInsertPayment Gives the user the permission to insert a payment for a volleyball transfer as the sender of the money.
This permission can be limited to some transfers for some users.
CanInsertPaymentAsFivb Gives the user the permission to insert a payment for a volleyball transfer, with access to all the fields of the payment.
This permission can be limited to some transfers for some users.
CanReleasePlayer Gives the user the permission to release a player from a volleyball transfer.
This permission can be limited to some transfers for some users.
CanSendAgainCompletedEMail Gives the user the permission to send again the e-mail that indicates that the transfer is completed.
This permission can be limited to some transfers for some users.
CanSendAgainConfirmEMail Gives the user the permission to send again the e-mail for the transfer confirmation.
This permission can be limited to some transfers for some users.
CanSendAgainReleaseConfirmedEMail Gives the user the permission to send again the e-mail when a transfer release is confirmed.
This permission can be limited to some transfers for some users.
CanSendAgainReleaseInitiatedEMail Gives the user the permission to send again the e-mail when a transfer release is initiated.
This permission can be limited to some transfers for some users.
CanSignAsSupervisor Gives the user the permission to sign a volleyball transfer as the supervisor.
This permission can be limited to some transfers for some users.
CanSignClubFrom Gives the user the permission to sign a volleyball transfer as the club of origin.
This permission can be limited to some transfers for some users.
CanSignClubTo Gives the user the permission to sign a volleyball transfer as the club of destination.
This permission can be limited to some transfers for some users.
CanSignFedFrom Gives the user the permission to sign a volleyball transfer as the national federation of origin.
This permission can be limited to some transfers for some users.
CanSignFedTo Gives the user the permission to sign a volleyball transfer as the national federation of destination.
This permission can be limited to some transfers for some users.
CanSignPlayer Gives the user the permission to sign a volleyball transfer as the player.
This permission can be limited to some transfers for some users.
CanSignRelease Gives the user the permission to sign the release of a volleyball transfer.
This permission can be limited to some transfers for some users.
CanSignReleaseAsClubTo Gives the user the permission to sign the release of a volleyball transfer as the club of destination.
This permission can be limited to some transfers for some users.
CanSignReleaseAsPlayer Gives the user the permission to sign the release of a volleyball transfer as the player.
This permission can be limited to some transfers for some users.
