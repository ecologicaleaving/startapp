Press Release Text

Represents a text for a press release.

The name of the XML element for a press release text is <PressReleaseText>.
The name of the XML element for a list of press release texts is <PressReleaseTexts>.

 Fields

The following table lists all the fields for a press release text.

Name Type Description
Accepted Boolean Indicates whether it is accepted.
AcceptedDT DateTime Date and time when it was accepted.
AcceptedUsername String Username of the user who has accepted it.

This field may contain formatting elements expressed as HTML codes.

This field may contain formatting elements expressed as HTML codes.
Keywords String Keywords.

Language String

Currently, the following languages are used in press release texts:

Code Language
ar العربية (Arabic)

en English (English)
es español (Spanish)
fr français (French)
it italiano (Italian)
nl Nederlands (Dutch)
pl polski (Polish)
pt português (Portuguese)
ru русский (Russian)

Almost all of the press release have an English version.
Only some of them are translated to other languages.

This is not the number of the press release.
