Beach Match Result Type

Type of result for a beach volleyball match.

 Values

The following table lists the values for the result types for a beach volleyball match.

Value

 Name Description

0

 Normal The match ends normally.

1

 ForfeitA Forfeit for team A.

2

 ForfeitB Forfeit for team B.

3

 ForfeitBoth Forfeit for both teams.

4

 InjuryA Injury for team A.

5

 InjuryB Injury for team B.

6

 InjuryBoth Injury for both teams.

7

 OutA Team A out of tournament.

8

 OutB Team B out of tournament.

9

 OutBoth Both teams out of tournament.

10

 DisqualifiedA Team A has been disqualified.

11

 DisqualifiedB Team B has been disqualified.

12

 DisqualifiedBoth Both teams have been disqualified.
