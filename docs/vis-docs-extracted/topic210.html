XSD Schemas

The following table lists all the XSD schemas related to the VIS Web Service.

Name Description
BankAccount List of bank accounts and their use.
BeachLive Beach volleyball live data.
BeachMatchLiveScore Live score of a beach volleyball match.
BeachMatchLiveScoreUpload Upload for the live score of a beach volleyball match.
BeachRoundRankingMethod Ranking method for a round in a beach volleyball tournament.
BeachTeamType Type for a team in a beach volleyball tournament.
BeachTournamentPrizeMoney Prize money for a beach volleyball tournament.
DateList List of dates.
DocumentList List of documents.
EmailAddresses List of e-mail addresses.
EventContent Content of an event.
EventVenues List of venues for event.
FaxNumbers List of fax numbers.
MedicalPracticeAuthorizations List of practice authorizations for a medical person.
MedicalPublications List of publications for a medical person.
MobileNumbers List of mobile numbers.
PersonLanguageList List of languages for a person.
PhoneNumbers List of phone numbers.
PresenceList List of presence for a person.
Schools List of followed schools.
VolleyLive Volleyball live data.
VolleyLiveUpload Upload for volleyball live data.
