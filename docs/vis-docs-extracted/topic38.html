Pool Ranking Rank List

Represents the ranking of the teams eliminated in a pool or a round.

Is used both for beach volleyball rounds and volleyball pools.

The list is simply a suite of entries. Each entry is returned by the web service as a JSON value or an XML element, depending on the desired output format. The name of the JSON value or XML element is Rank.

The following table lists the properties of an entry.

Name Type Description
RoundRank Int32 The rank of the team in the pool or round.
TournamentRank Int32 The rank of the team in the tournament.
