Discrete values > Media > Media Person Function
Media Person Function

Function for a media person.
The values are OR-ed to have the final value, which is a 32-bits integer.

 Values

The following table lists the values for the available functions for a media person.
Note that the Value column contains hexadecimal values.

Value Name Description
0x0001 ConfedFedPressOfficier Confederation or national federation press officer.
0x0002 Editor Editor.
0x0004 JournalistReporter Journalist / Reporter.
0x0008 Photographer Photographer.
0x0010 RadioCommentator Radio commentator.
0x0020 RadioTechnician Radio technician.
0x0040 TVCommentator TV commentator.
0x0080 TVTechnician TV technician.
