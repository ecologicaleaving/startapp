Beach Volleyball Teams Filter

Filter for a list of beach volleyball teams.

 Parameters

The following table lists the attributes for the filter for beach volleyball teams.

Name Type Description
HasPlayed Boolean

Filtering about whether the team has played the tournament.

The value of this parameter can be one of the following:

Value Meaning

Empty
OR
not specified

 The filter does not use the fact that the team has played or not in the tournament.
false The resulting list will contain only teams that have not played in the tournament.
true The resulting list will contain only teams that have played in the tournament.
NoPlayer Int32 Number of the player to look.
If this attribute is specified, the attributes NoPlayer1, NoPlayer2, NoTeam and NoTournament should not be specified.
If this attribute is specified, it is recommended to set the HasPlayed attribute to true to not receive the future tournaments and those in which the team has been deleted and is withdrawned.
NoPlayer1 Int32 Number of the first player of a team to look.
If this attribute is specified, the attribute NoPlayer2 must be specified.
If this attribute is specified, the attributes NoPlayer, NoTeam and NoTournament should not be specified.
If this attribute is specified, it is recommended to set the HasPlayed attribute to true to not receive the future tournaments and those in which the team has been deleted and is withdrawned.
NoPlayer2 Int32 Number of the second player of a team to look.
If this attribute is specified, the attribute NoPlayer1 must be specified.
If this attribute is specified, the attributes NoPlayer, NoTeam and NoTournament should not be specified.
If this attribute is specified, it is recommended to set the HasPlayed attribute to true to not receive the future tournaments and those in which the team has been deleted and is withdrawned.
NoTeam Int32 Number of the team to look.
If this attribute is specified, the attributes NoPlayer, NoPlayer1, NoPlayer2 and NoTournament should not be specified.
If this attribute is specified, it is recommended to set the HasPlayed attribute to true to not receive the future tournaments and those in which the team has been deleted and is withdrawned.
NoTournament Int32 Unique number of the tournament.
If this attribute is specified, the attributes NoPlayer, NoPlayer1, NoPlayer2 and NoTeam should not be specified.
PlaysInConfederationQuota String The code of the confederation for teams that play in a confederation quota selection round.
When this attribute is specified, the filtered teams will contains only those that plays a confederation quota round for the specified confederation.
If the attribute value is empty, the filter will select all the teams that have played in any confederation quota selection round.
PlaysInFederationQuota String The code of the federation for teams that play in a federation quota selection round.
When this attribute is specified, the filtered teams will contains only those that plays a federation quota round for the specified federation.
If the attribute value is empty, the filter will select all the teams that have played in any federation quota selection round.
PlaysInMainDraw Boolean

Filtering about the participation of a team in the main draw.

The value of this parameter can be one of the following:

Value Meaning

Empty
OR
not specified

 The filter does not use the main draw participation.
false The resulting list will contain only teams that do not participate in the main draw.
true The resulting list will contain only teams that participate in the main draw.
PlaysInQualification Boolean

Filtering about the participation of a team in the qualification tournament.

The value of this parameter can be one of the following:

Value Meaning

Empty
OR
not specified

 The filter does not use the qualification tournament participation.
false The resulting list will contain only teams that do not participate in the qualification tournament.
true The resulting list will contain only teams that participate in the qualification tournament.
Status String

Filtering about the status of the team.
If this attribute is specified, it must contains a list of beach volleyball team statuses separated with a space. Each status in the list can be specified by its value or Name.
Type String Filtering about the typeof the team.
If this attribute is specified, it must contains a list of beach volleyball team types separated with a space. Each typein the list can be specified by its value or Name.
 Errors

When the filter is used, the following errors may be returned by the web service:

Error Description
<BadParameter>
  • The HasPlayed attribute is not a valid Boolean value.
  • The NoPlayer attribute is not a valid integer value.
  • The NoPlayer1 attribute is not a valid integer value.
  • The NoPlayer2 attribute is not a valid integer value.
  • The NoTeam attribute is not a valid integer value.
  • The NoTournament attribute is not a valid integer value.
  • The PlaysInMainDraw attribute is not a valid Boolean value.
  • The PlaysInQualification attribute is not a valid Boolean value.
<ParameterMissing>
  • The NoPlayer1 attribute is specified, but not the NoPlayer2 attribute.
  • The NoPlayer2 attribute is specified, but not the NoPlayer1 attribute.
