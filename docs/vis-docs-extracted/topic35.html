Beach Volleyball Team

Represents a team in a beach volleyball tournament.

A beach volleyball team consists of 2 players of the same federation, without substitutes.

The name of the XML element for a beach volleyball team is <BeachTeam>.
The name of the XML element for a list of beach volleyball teams is <BeachTeams>.

The following table lists the valid values for this field.

Value

 Description

<= -25

This result should not be used for entry points ranking.

-2

This result should not be used for entry points ranking.

0

 Team has not played the tournament.

 Real rank.
