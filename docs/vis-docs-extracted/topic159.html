Request to get a beach volleyball Olympic Selection ranking
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetBeachOlympicSelectionRanking"
         Gender="<gender>"
         GamesYear="<year of the Olympic Games>"
         OnlySelected="<boolean value>"
         ReferenceDate="<ranking reference data>"
         Fields="list of the fields to return" />

Where:

Attribute Type Description
GamesYear Int32 Year of the Olympic Games.
If there is no Olympic Games at the specified year, the returned ranking will be empty.
If this parameter is not specified, the web service will return the ranking for the latest Olympic Games with a ranking.
Gender Gender Gender of the teams to retrieve the ranking for.
This is a mandatory parameter.
OnlySelected Boolean

Indicates the selection status of the returned teams.

The value of this parameter can be one of the following:

Value Description

Empty
OR
not specified

 The resulting list will contain both selected and not-selected teams.
false The resulting list will contain only non-selected teams.
true The resulting list will contain only selected teams.
ReferenceDate Date Reference date of the ranking: when the ranking has been calculated.
If there is no Olympic Games selection ranking calculated at the specified date, the returned ranking will be empty.
If this parameter is not specified, the web service will return the ranking for the latest reference date.

If both GamesYear and ReferenceDate are not specified, the web service will first calculate the GamesYear value from the existing rankings and then the ReferenceDate value.

 Response
When the request is successful, the service will return the a <BeachOlympicSelectionRanking> XML element that contains beach Olympic selection ranking entries.
 Errors

In addition to global errors, the following errors can be sent by the server:

Error Description
<BadParameter>
  • The GamesYear parameter, if specified, is not a valid 32-bits integer number.
  • The Gender parameter has not a valid value.
  • The OnlySelected parameter, if specified, is not a valid Boolean.
  • The ReferenceDate parameter, if specified, is not not a valid date.
<ParameterMissing> You must specify the Gender parameter.
 Security

This is a public request: any client can retrieve information.

All the fields for a beach volleyball Olympic selection ranking are public.

 Example

Retrieve the latest Olympic Games selection ranking for women.

Web service request
Copy Code 
<Request Type="GetBeachOlympicSelectionRanking"
         Gender="W"
         Fields="Position TeamName TeamCountryCode NbParticipations SelectionRank Points Status" />
