Volley Statistic

Represents a volleyball statistic.

The name of the XML element for a statistic is <VolleyStatistic>.
The name of the XML element for a list of statistics is <VolleyStatistics>.

The statistics calculated by the web service can be for players or for teams, in addition, the caller can ask for the sum of all players in a team. The web service keeps the statistics for each set of a match and the caller can ask for the statistics by set, the sum by match or the sum by tournament.

The relation is missing (null in JSON) if the statistic is for a player.
Tournament VolleyTournament The tournament.
 Side-out and Rally Point systems

Initially, the points were counted using the side-out system: when a team wins a rally, it won a point only if serving; otherwise, it won the right to serve.

Later, starting in 1998, the rule change to the Rally Point system: when a team wins a rally, it won a point.

The properties with the name ending with SideOut are only available for statistics of old volleyball tournaments.
