VBA statistics file (for shirt numbers up to 99)

This is a preliminary version of the documentation and there can remain errors on it.

If you find errors (even typos) in this documentation, please contact the FIVB.

The VBA statistics file is named Statxxx.i, where xxx is the number of the match in the tournament, using 3 digits, and i is 1 for the first acquisition PC and 2 for the second one.

This page describes the format of the file when used with the Match format 4, enabling up to 99 for the shirt numbers.

The content of the file is a collection of entries. The available entries can be identified by bits 0 to 4 of their first byte. Normally, these bits contain the shirt number of the player, but can contain special values to identify the following data.

The different entries for the file are listed in the following sections.

 Action

An action is defined into four 8-bits integers.

The format of an action is the following:

  • Byte 1 contains the shirt number. It can be 1 to 99. Some special values are defined, see the table below for special values.
  • Byte 2 contain the skill, see the table below for the values.
  • Byte 3 contain the note, see the table below for the values.
  • Byte 4 is 0 for team A or 1 for team B.

 End of set (version 2)

An end of set is coded with four 8-bits values:

  • Byte 1 contains the special code: 252 (0xFC).
  • Byte 2 contains the number of the set.
  • Byte 3 contains the number of points of team A.
  • Byte 4 contains the number of points of team B.
 Libero substitution

A libero substitution is coded with four 8-bits values:

  • Byte 1 contains the special code: 250 (0xFA).
  • Byte 2 contains the shirt number of the libero leaving the court.
  • Byte 3 contains the contain the shirt number of the libero entering the court.
  • Byte 4 is 0 for team A or 1 for team B.
 New line-up

A new line-up is coded with eight 8-bits values.

  • Byte 1 contains the special code: 254 (0xFE).
  • Byte 2 is 0 for team A or 1 for team B.
  • Byte 3 is the shirt number of the server (back right player).
  • Byte 4 is the shirt number of the front right player.
  • Byte 5 is the shirt number of the front middle player.
  • Byte 6 is the shirt number of the front left player.
  • Byte 7 is the shirt number of the back left player.
  • Byte 8 is the shirt number of the back middle player.
 Player substitution

A player substitution is coded with four 8-bits values:

  • Byte 1 contains the special code: 253 (0xFD).
  • Byte 2 contains the shirt number of the player leaving the court.
  • Byte 3 contains the contain the shirt number of the player entering the court.
  • Byte 4 is 0 for team A or 1 for team B.
