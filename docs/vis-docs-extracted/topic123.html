Volley Match Status

Status for a volleyball match.

The statuses are defined in the order they are found in a match, so you can use the <, <=, ==, !=, > and >= operators between two values to check where we are in a match.

There is no status to indicate that the set 7 is finished, because a match can have at most seven sets and, if there is a seventh set, the end of set 7 is the end of the match.

If the match is played in one set, the status following InSet1 is Finished.
If the match is played in three sets, the status following InSet3 is Finished.
If the match is played in five sets, the status following InSet5 is Finished.

If a match is played in three sets and a team wins the first two sets, the status following InSet2 is Finished.
If a match is played in five sets and a team wins after 3 sets, the status following InSet3 is Finished. If a team wins after 4 sets, the status following InSet4 is Finished.
If a match is played in seven sets and a team wins after 4 sets, the status following InSet4 is Finished. If a team wins after 5 sets, the status following InSet5 is Finished. If a team wins after 6 sets, the status following InSet6 is Finished.
