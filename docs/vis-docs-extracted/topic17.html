Date and Time

A date and time is expressed using the standard XML format for date and time: YYYY‑MM‑DDThh:mm:ss.

Where:

Part Description
YYYY A 4 digits year.
MM A 2 digits month number.
January is 01 and December is 12.
DD A 2 digits day number.
hh A 2 digits number of hours, using a 24 hours clock.
mm A 2 digits number of minutes.
ss A 2 digits number of seconds.

Note that there can be a Z at the end of the string, meaning that the date and time is in UTC (Universal Time Coordinate, also called GMT) format.
