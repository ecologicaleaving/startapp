Volleyball Pool Technical Info

Represents technical information about a volleyball pool.

The technical information is used to save information about the ranking of the pool and the ranking of the teams eliminated in the pool.

 Properties

The following table lists all the properties of the technical information.

Name Type Description
NoPools Int32 set

The numbers of the pools to add to the current one for ranking.
The value is missing if the ranking is made only with the matches of the pool.

Some play-off pools are ranked using the result of the pools and also the result of the matches in elimination pools. In any case, the ranking will only contains the teams defined in NoTeams.
NoTeams Int32 set

The numbers of the teams to include in the ranking.
The value is missing if the ranking must contain all the teams that play in the pool.
Ranks PoolRankingRankList

The ranks of the teams that are eliminated in the pool.
This content is only available if teams are eliminated in the pool.
