City optional xs:string
City. The attribute is omitted if the city is unknown.
CountryCode optional xs:string
Code of the country. The code is omitted if the country is unknown.
EndDate optional xs:date
End date. The attribute is omitted if the end date is unknown.
Gender optional Restriction of EventGender Simple Type
Gender.
Logos optional xs:string
The logos to use. The attribute is empty if there is no logos. The value is an XML element that defines the attributes. See the documentation (http://www.fivb.org/VisSDK/VisWebService/#Event%20Logos.html) for the format of the XML element.
Name optional xs:string
Name. The attribute is omitted if the name is unknown.
No required xs:positiveInteger
Number.
NoEvent optional xs:positiveInteger
Number of the event. The attribute is omitted if the tournament doesn't belong to an event.
PoolName optional xs:string
Name of the pool. This attribute is used when an event, such as the World League or the Grand Prix, contains more than one tournament. It defines the name of the pool of the tournament in the event.
RoundCode optional xs:string
Code of the round. This attribute is used when an event, such as the World League or the Grand Prix, contains more than one tournament. It defines the code of the round of the tournament in the event.
Season optional xs:string
Season. The attribute is omitted if the season is unknown.
ShortName optional xs:string
Short name. The attribute is omitted if the short name is unknown
ShortNameOrName optional xs:string
The short name or the name. The attribute is omitted if there is no short name and no name. If the short name is defined, this is the short name; otherwise, this is the name.
StartDate optional xs:date
Start date. The start date is omitted if the start date is unknown
