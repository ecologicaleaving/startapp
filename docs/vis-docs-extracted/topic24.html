Language Code

A language is specified by a unique code for each culture, based on RFC 4646. The code is a combination of an ISO 639-1 two-letter lowercase culture code associated with a language and an ISO 3166-1-alpha-2 uppercase subculture code associated with a country or region.
The format for the code is "<languagecode>-<country/regioncode>", where <languagecode> is the language code and <country/regioncode> is the subculture code. Examples include ja-JP for Japanese (Japan) and en-US for English (United States). In cases where a two-letter language code is not available, a three-letter code derived from ISO 639-2 is used.
Note that some codes also specify an ISO 15924 script. For example, "-Cyrl" specifies the Cyrillic script and "-Latn" specifies the Latin script. A code including a script is rendered using the pattern <languagecode>-<scripttag>-<country/regioncode>. An example of this type of code is uz-Cyrl-UZ for Uzbek (Cyrillic, Uzbekistan). A neutral code is specified by only the two-letter lowercase language code. For example, "fr" specifies the neutral code for French, and "de" specifies the neutral code for German.
