Overview
ChallengeRequest Extension of Challenge Complex Type
Challenge asked by a team.
NoTeam required xs:positiveInteger
Number of the team that asks the challenge. The attribute is omitted if the challenge is from a referee or the team number is unknown.
PointsTeamA required xs:nonNegativeInteger
Number of points for team A, at the time of the challenge. The attribute is missing if the points for team A are unknown.
PointsTeamB required xs:nonNegativeInteger
Number of points for Team B, at the time of the challenge. The attribute is missing if the points for team B are unknown.
RequestedFrom optional Restriction of ChallengeRequestedFrom Simple Type
Who has requested the challenge. The attribute is empty if the requester is unknown.
Type optional Restriction of ChallengeRequestType Simple Type
Type of the challenge. If the attribute is missing the type is unknown.
Identifier optional UUID Simple Type
Identifier of the event. This is a UUID.
Text optional xs:string
Additional information about the challenge. The attribute is omitted if there is no additional information.
TimeOffset optional TimeOffset Simple Type
Time offset, from the start of the match.
