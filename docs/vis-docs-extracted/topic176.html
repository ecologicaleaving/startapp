Request to get a list of press releases
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetPressReleaseList"
         Number="10"
         StartIndex="11"
         Fields="<list of the fields to return>">
  <Filter /> <!-- Optional: contains the filter to use -->
</Request>

Where:

Attribute Description
Number Number of press releases to return.
If it is not specified, all the press releases that conform to the filter are returned.
Also, if this number is specified, the press releases will be sorted by their date and time and the latest are returned.
StartIndex Index of the first press release to return.
The index of the first press release is 1.
If it is not specified, the default value is 1, meaning that the web service will return the press releases starting at the first one.
Also, if this number is specified, the press releases will be sorted by their date and time and the latest are returned.

Filter

The filter is optional.
If it is not specified, the response will contain all the press release texts.
Please see PressReleaseFilter for more information.

Fields

The list of fields is mandatory.
It can contain all the fields in the PressRelease data. Only the fields you have access to will be returned.

 Response
When the request is successful, the filter will return the list of filtered press releases.
 Security

This is a public request: any client can retrieve information.

Some of the fields for a press release are public and other are not.
The response from the server will contain only the fields the client has access to.

 Example

Retrieve the last 10 press releases, for World League 2012 (WL2012) or World Grand Prix 2012 (WGP2012).

Web service request
Copy Code 
<Request Type="GetPressReleaseList"
         Number="10"
         Fields="DateTime Category SpecialType TournamentCode Languages">
  <Filter TournamentCodes="WL2012 WGP2012" />
</Request>
