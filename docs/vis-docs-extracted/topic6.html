VIS web service documentation
Introduction

This documentation describes the communications between the VIS Web Service and a client application.

The VIS Web Service enables third party applications to access all public data about volleyball, beach volleyball and other events.
For registered user, access to more data and to data modification is also possible.

If you are developing a .NET Windows application, or need to access to the VIS Web Service from another web site in ASP.NET, you can use the Fivb.Vis.DataWPF assembly that contains the definition of all types and requests used by the VIS Web Service.
For other platforms, you currently need to write yourself the code to communicate with the VIS Web Service.

The web service is evolving to add new features, you can see the release notes of the different versions at the following URL: https://www.fivb.org/VisSDK/#Release Notes.html
