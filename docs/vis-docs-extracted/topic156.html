Filters

Some requests that ask for a list of data can contain a filter to limit the results sent back from the server.

A filter has the following standard format:

Filter format
Copy Code 
<Filter optional-parameters>
  <!-- Filter data -->
</Filter>

For some filter, it is possible to add tags to the filter, as explained in the Tags Filtering page.
