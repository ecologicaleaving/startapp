Get request
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetArticle"
         No="<artcle number>"
         Fields="<optional: list of the fields to return>" >
  <Relation Name="<name of the relation>" Fields="<list of the fields for the relation>">/> <!-- optional: contains a relation (can be used multiple times) -->
</Request>

Where:

Attribute Description
No The number of the article.
This is a mandatory parameter.

Fields

The list of fields is optional: if it not specified, all the fields you have access to will be sent.
It can contain all the fields in the Article data. Only the fields you have access to will be returned.

Relations

The relations are optional. They permit to get additional information about the article, such as the texts or the tags.
A list of the available relations for an article can be found here.

 Response
When the request is successful, the service will return the data of an article.
 Security

This is a public request: any client can retrieve information.

Some of the fields for an article are public and other are not.
The response from the server will contain only the fields the client has access to.

 Example

Retrieve the article with the number 28639.

Web service request
Copy Code 
<Request Type="GetArticle" No="28639"
         Fields="No Category Source SourceCategory TeamCode TournamentCode NoMatch PublishOnHome VideoUri IsVideoLive">
  <Relation Name="Texts" Fields="Accepted LanguageCode LanguageEnglishName LanguageNativeName AuthorName Headline Introduction PhotoTitle Quotes Text" />
  <Relation Name="Tags" Fields="TagText TagUserText" />
</Request>
