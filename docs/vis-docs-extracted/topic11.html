Base Types

For all data, an empty value means that there is no value. This is equivalent to the NULL SQL value.

The following type list all the base types for the web service.

Name Description
Amount A money amount.
Boolean A boolean value: false or true.
Confederation code A confederation code.
Country code A code of country.
Date A date.
Date and time A date and a time.
Duration A duration.
Federation code A national federation code.
Humidity A humidity.
Int32 A 32-bits integer number.
Int64 A 64-bits integer number.
Length A length.
Speed A speed.
Time A time.
Weight A weight.
