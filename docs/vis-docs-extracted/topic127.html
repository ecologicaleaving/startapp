Volleyball Tournament Action

Available actions for a volleyball tournament.
The values are OR-ed to have the final value, which is a 32-bits integer.

 Values

The following table lists the values for the available actions for a volleyball tournament.
Note that the Value column contains hexadecimal values.

Value

 Name Description

0x0001

 Administrator An administrator has all accesses.

0x0002

 CanEditTeamData A federation can edit teams data.

0x0004

 CanSelectPlayers A federation can select the players.

0x0008

 CanUpdatePlayers A federation can update players.

0x0010

 CanEditRoster A federation can edit the roster.

0x0020

 CanSelectFinalPlayers A federation can select final players.

0x0040

 CanEditShirtColors A federation can edit shirt colors.

0x0080

 CanPrintO2 A federation can print the O-2 report.

0x0100

 CanPrintO2bis A federation can print the O-2bis report.

0x0200

 CanSelectCaptainLiberos A federation can select the captain and liberos.

0x0400

 CanPrintO2A A federation can print the O-2A report.
0x0800 CanPreselectPlayers A federation can preselect the players.
0x1000 TeamsArePublic The teams details are public.
