Event Logos

Logos for an event.

When the logos are in an XML attribute, they are formatted as an XML element. When the logos are exported in JSON, they are formatted as a JSON value.

 Properties

The following table lists all the properties for the logos.

Name Type Description
Competition Int32 The number of the logo for the competition.
The attribute is missing if the logo is not defined.
Deprecated: if one of the other CompetitionXXX property is available, use it instead.
CompetitionHighRes Int32 The number of the high-resolution image to use as the competition logo.
The attribute is missing if the logo is not defined. In this case, you can try to use another version of the logo.
CompetitionIcon Int32 The number of the image to use as the competition icon.
The attribute is missing if the icon is not defined.
The icon is intended to be used on web sites, as the page icon.
CompetitionMobile Int32 The number of the image to use as the competition logo for mobile applications.
The attribute is missing if the logo is not defined. In this case, you can try to use another version of the logo.
CompetitionWeb Int32 The number of the image to use as the competition logo for the web.
The attribute is missing if the logo is not defined. In this case, you can try to use another version of the logo.
Organizer Int32 The number of the logo to use as the competition organizer.
The attribute is missing if the logo is not defined.
Publicity Int32 The number of the logo to use for the publicity.
The attribute is missing if the logo is not defined.

 To get an image, you must the GetImage.aspx URL.
