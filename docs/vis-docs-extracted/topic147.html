Filter for events

Filter for events.

 Attributes

The following table lists the attributes for the filter for events.

Name Type Description
FirstDate Date First date of the events to select.
When this attribute is specified, the filtered events will contain only the events that ends after the date specified in the parameter .
HasBeachTournament Boolean

Filter about HasBeachTournament.

The value of this attribute can be one of the following:

Value Description

Empty
OR
not specified

 The filter does not use the HasBeachTournament field.
false The resulting list will contain only events that have no beach volleyball tournaments.
true The resulting list will contain only events that have at least one beach volleyball tournaments.
HasMenTournament Boolean

Filter about HasMenTournament.

The value of this attribute can be one of the following:

Value Description

OR
not specified

 The filter does not use the HasMenTournament field.
false The resulting list will contain only events that have no tournaments for men.
true The resulting list will contain only events that have at least one tournament for men.

IsVisManaged Boolean

Filter about IsVisManaged.

The value of this parameter can be one of the following:

Value Description

Empty
OR
not specified

 The filter does not use the IsVisManaged field.
false The resulting list will contain only events that are not managed with the VIS system.
true The resulting list will contain only events that are managed with the VIS system.
LastDate Date Last date of the events to select.
When this attribute is specified, the filtered events will contain only the events that starts before the date specified in the attribute.
NoParentEvent Int32 Number of the parent event for the events to select.
When this attribute is specified, the filtered events will contain only the events that have the event with the specified number as a parent.
To select the top-level events, use this attribute with the value 0.
OpenForAccreditation Boolean

Filter about OpenForAccreditation.

The value of this parameter can be one of the following:

Value Description

Empty
OR
not specified

 The filter does not use the check whether the event is opened for accreditation.
false The resulting list will contain only events that are not opened for accreditation.
true The resulting list will contain only events that are opened for accreditation.

An event is opened for accreditation if the AccreditationEndDate and AccreditationStartDate properties are defined and the current date is between these two dates.
Types EventTypeSet Set of the types of the events to select.
If this attribute is null, the events will not be selected by their type.
