Request to get the ranking of a beach volleyball round
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetBeachRoundRanking"
        ; No="<round number>"
        ; Fields="<list of the fields to return>" />

Where:

Attribute Description
No Number of the beach volleyball round.
This is a mandatory parameter.

Fields

The list of fields is mandatory.
It can contain all the fields in the Beach Volleyball Round Ranking Entry data. All the fields are public, so you can request any one.

 Response
When the request is successful, the service will return the a <BeachRoundRanking> XML element that contains beach volleyball round ranking entries.
 Errors

In addition to global errors, the following errors can be sent by the server:

Error Description
<BadParameter> The No parameter is not a valid 32-bits integer number.
<NoData> The beach volleyball round with the specified number does not exist or has been deleted.
<NotARankingRound> The beach volleyball round has not a ranking: it uses direct elimination.
<ParameterMissing> You must specify the No parameter.
 Security

This is a public request: any client can retrieve information.

All the fields for a beach volleyball round ranking are public.

 Example

Retrieve the ranking of the round with the number 980.

Web service request
Copy Code 
<Request Type="GetBeachRoundRanking"
         No="980"
         Fields="Position Rank TeamFederationCode TeamName MatchPoints MatchesWon MatchesLost" />

The response is:

Web service response
Copy Code 
<BeachRoundRanking>

  <BeachRoundRankingEntry MatchesLost="1" MatchesWon="2" MatchPoints="5" Position="2" Rank="2" TeamFederationCode="AUT" TeamName="Doppler-Mellitzer"/>
  <BeachRoundRankingEntry MatchesLost="2" MatchesWon="1" MatchPoints="4" Position="3" Rank="3" TeamFederationCode="USA" TeamName="Gibb-Rosenthal"/>
