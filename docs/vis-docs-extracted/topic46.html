Article Text
This documentation is preliminary, as the articles API is currently in development. The information here is subject to changes.

Represents a text for an article.

This data type supersedes the PressReleaseText data type.

The name of the XML element for a text is <ArticleText>.
The name of the XML element for a list of texts is <ArticleTexts>.

 Properties

The following table lists all the properties for a text.

Name Type Description
AcceptedAt DateTime The date and time when it was accepted.
AcceptedUsername String The username of the user who has accepted it.
AdditionalInformation String Additional information.
This field may contain formatting elements expressed as HTML codes.
AuthorName String The name of the author.
Headline String The headline.
Introduction String The introduction.
This field may contain formatting elements expressed as HTML codes.
IsAccepted Boolean Indicates whether it is accepted.
LanguageCode LanguageCode

The code of the language for the text.

Currently, the following languages are used in articles:

Code Language
ar العربية (Arabic)
de Deutsch (German)
en English (English)
es español (Spanish)
fr français (French)
id bahasa Indonesia (Indonesian)
it italiano (Italian)
ja 日本語 (Japanese)
nl Nederlands (Dutch)
pl polski (Polish)
pt português (Portuguese)
ru русский (Russian)
zh-Hans 简化字 (Simplified Chinese)


Almost all of the articles have an English version.
Only some of them are translated to other languages.
LanguageEnglishName String The name of the language, in English.
LanguageNativeName String The native name of the language.
LastChangeDT DateTime The date and time of the last change in the text.
LastChangeUser Int32 The number of the user who has made the last change in the text.
LastChangeUsername String The username of the user who has made the last change in the text.
No Int32 The unique number of the text.
This number is unique among all the texts.
This is not the number of the article.
NoArticle Int32 The number of the article to which the text is related to.
PhotoTitle String The title for the photo, if any.
Quotes String The quotes.
Text String The text.
This field may contain formatting elements expressed as HTML codes.
Version Int32 The version of the text.
