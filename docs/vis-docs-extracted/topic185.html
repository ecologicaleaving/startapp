Request to get a list of volleyball matches
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetVolleyMatchList"
         Fields="<list of the fields to return>">
         Version="<version of local list>">
  <Filter /> <!-- optional: contains the filter to use -->
</Request>

Fields

The list of fields is mandatory.
It can contain all the fields in the VolleyMatch data. Only the fields you have access to will be returned.

Filter

The filter is optional.
If it is not specified, the response will contain all the matches.
Please see the VolleyMatchFilter documentation for more information.

Version

The version is optional.
The goal of this parameter is to request an update to an existing local list: for the first request, you don't specify the parameter (or set it to 0) and you receive the full list of the filtered matches; for subsequent requests, you specify the version of the local list and the web service only returns the changes (new matches or matches that have changed). Doing this will decrease the bandwidth used by the local application.
Note that the version of all the matches (not only the filtered ones) is returned in the root of the response.

 Response

When the request is successful, the web service returns the filtered list of matches.

In addition, the root element contains the number of matches that are returned and the version of all the matches .

 Security

This is a public request: any client can retrieve information.

Some of the fields for a match are public and other are not.
The response from the web service will contain only the fields the client has access to.

 Example

Get the list of the matches in the tournament with the number 979, that are played in July.

Web service request
Copy Code 
<Request Type="GetVolleyMatchList" Fields="No NoInTournament DateTimeLocal DateTimeUtc City Hall MatchPointsA MatchPointsB">
  <Filter FirstDate="2017-07-01" LastDate="2017-07-31" NoTournament="979" />
  <Relation Name="TeamA" Fields="Code Name" />
  <Relation Name="TeamB" Fields="Code Name" />
</Request>
