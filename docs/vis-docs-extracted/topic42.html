Tag Text
This documentation is preliminary, as the tags API is currently in development. The information here is subject to changes.

Represents the text of a tag, in a specific language.
Each tag can be written using different text, for different languages.

The name of the XML element for a text is <TagText>.
The name of the XML element for a list of texts is <TagTexts>.
