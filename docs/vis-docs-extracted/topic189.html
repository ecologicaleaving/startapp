Request to get the ranking of the players in a volleyball tournament
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetVolleyPlayersRankingList"
         No="<the number of the ranking>"
         NoTournament="<the number of the tournament>"
         NumberOfPlayers="<the number of players to return>";
         NumberOfScorers="<the number of scorers to return>";
         Skills="<the skills to return>" />

No and NoTournament

One of these parameters must be specified. If both are specified, the web service uses the No parameter.

Each tournament can have zero, one or many players' ranking. For example, there can be a ranking for the preliminary round, the final round and a global ranking.

To get the default players' ranking for a tournament, just specify the number of the tournament with the NoTournament parameter.
To get a specific players' ranking, specify the ranking with the No parameter.

NumberOfPlayers / NumberOfXxx

Indicates how many players must be returned by the web service, for each skill. If the parameter is missing, the web service returns all the players for each ranking. If there is less players in the ranking for a skill, only the existing players are returned.

NumberOfPlayers is the default value for all the skills. If you want a different number of players for the scorers, you must also use the NumberOfScorers parameter.

Skills

Indicates which skills must be returned by the web service. If the parameter is missing, the web service returns all the skills.

The parameter contains a list of skills separated by a space.

The following skills are available:

Name Description
Block Block.
Dig Dig.
Libero Ranking for the liberos.
Reception Reception.
Scorer Ranking for the scorers (points marked with Block, Serve and Spike).
Service Service.
Set Set.
Spike Spike.
 Response

When the request is successful, the web service returns the requested players' rankings.

 Errors

In addition to the standard errors, the following errors can be returned.

Code Description
NoDefaultRanking The specified tournament has no default ranking.
NoRanking The specified tournament has no ranking.
ParameterMissing The No or NoTournament parameter must be specified.
 Security

This is a public request: any client can retrieve information.

All the properties are public and are returned by the web service.

 Example

Get the 10 best players of tournament 955, for specified skills.

Web service request
Copy Code 
<Request Type="GetVolleyPlayersRankingList"
         NoTournament="955"
         NumberOfPlayers="10"
         Skills="Set Spike Libero Scorer" />
