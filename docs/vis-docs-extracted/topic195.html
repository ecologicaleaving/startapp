Request to get a volleyball tournament
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetVolleyTournament"
         No="<tournament number>">
         Fields="<Optional: list of the fields to return>" />

Where:

Attribute Description
No The number of the volleyball tournament.
This is a mandatory parameter.

Fields

The list of fields is optional: if it not specified, all the fields you have access to will be sent.
It can contain all the fields in the VolleyTournament data. Only the fields you have access to will be returned.

 Response
When the request is successful, the service will return the data of a volleyball tournament.
 Errors

In addition to global errors, the following errors can be sent by the server:

Code Description
BadParameter The No parameter is not a valid 32-bits integer number.
NoData The volleyball tournament with the specified number does not exist or has been deleted.
ParameterMissing You must specify the No parameter.
 Security

This is a public request: any client can retrieve information.

Some of the fields for a volleyball tournament are public and other are not.
The response from the web service will contain only the fields the client has access to.

 Example

Get the tournament with the number 979.

Web service request
Copy Code 
<Request Type="GetBeachTournament"
         No="979"
         Fields="Name City CountryCode StartDate EndDate Status" />

The response is (formatted for readability):

Web service response
Copy Code 
<VolleyballTournament No="979"
                      City="Various"
                      CountryCode=""
                      EndDate="2017-07-08"
                      Name="World League 2017 - Group 1"
                      StartDate="2017-06-02"
                      Status="3"
                      WebSite="https://worldleague.2017.fivb.com" />
