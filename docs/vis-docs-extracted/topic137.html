VIS web service documentation
Discrete Values

There are plenty of values, in the database, that have a special meaning.

Most of the values are defined with a never-changing value and a name. Some of them only have values. The server always sends the value to the client and, for the moment, accepts only values in the requests.

The following table lists all the discrete values for the VIS web service.
