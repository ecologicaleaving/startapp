Beach Volleyball Round Ranking Entry

Represents an entry in the ranking for a beach volleyball round.
Note that not all beach volleyball rounds have a ranking: most of them use direct elimination.

The name of the XML element for an entry for the ranking is <BeachRoundRankingEntry>.

 Fields

The following table lists all the fields in an entry for a beach volleyball round ranking.

Name Type Description
MatchesLost Int32 Number of matches lost by the team in the round.
MatchesWon Int32 Number of matches won by the team in the round.
MatchPoints Int32 Number of match points received by the team in the round.
The number of match points received for a winning or a losing match depends on the rules used in the tournament.
NoRound Int32 Unique number of the round to which the ranking entry belongs.
NoTeam Int32 Number of the team at this rank.
PointsLost Int32 Number of points lost by the team in the round.
PointsWon Int32 Number of points won by the team in the round.
Position Int32 Position of the team in the ranking.
Is always from 1 to the number of teams in the round.
This field must be used to sort the ranking in the right order.
Rank Int32

Rank of the team.
When there is a tie between two or more teams, they can have the same rank. In this case, the first team has a positive ranking and the other ones have a negative ranking with the same absolute value. The negative rank values should not be displayed to the user.

The following table lists the valid values.

Value Description

> 0

 Real rank of the team.

1001

 Eliminated in a confederation quota selection round or a federation quota selection round.
The rank must be displayed as N/A (not any) to the user.
This value is only used for confederation and federation quota selection rounds.

1002

 Qualified from a confederation and federation quota selection round to the qualification tournament.
The rank must be displayed as Q (qualified) to the user.
This value is only used for confederation and federation quota selection rounds.
SetsLost Int32 Number of sets lost by the team in the round.
SetsWon Int32 Number of sets won by the team in the round.
TeamFederationCode FederationCode Code of the federation for the team.
TeamName String Name of the team.
