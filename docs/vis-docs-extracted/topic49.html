Persons > Courses > Course
Course

Represents a course that a person can follow.

The name of the XML element for a course is <Course>.
The name of the XML element for a list of courses is <Courses>.

 Properties

The following table lists all the properties for a course.

Name Type Description
Code String The code.
DateFrom Date The date when the course starts.
DateTo Date The date when the course finishes.
Is empty when there is no end date.
DeletedDT DateTime Empty if the course is valid.
The date and time of the deletion when the course is deleted.
Description String The description.
LastChangeDT DateTime The date and time of the last change.
LastChangeUser Int32 The number of the user who has made the last change.
LastChangeUsername String The name of the user who has made the last change.
Name String The name.
No Int32 The unique number.
Version Int32 Version.
The version is global to all courses. It is used when requesting a list of courses, to get only the changes of the list.
