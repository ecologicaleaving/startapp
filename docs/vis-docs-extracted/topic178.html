Request to get a list of press release texts
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetPressReleaseTextList"
         Number="<number of press release texts to return>"
         StartIndex="<index of the first press release text to return>"
         Fields="<list of the fields to return>">
  <Filter /> <!-- Optional: contains the filter to use -->
</Request>

Where:

Attribute Description
Number Number of press release texts to return.
If it is not specified, all the press release texts that conform to the filter are returned.
Also, if this number is specified, the press release texts will be sorted by their date and time and the latest are returned.
StartIndex Index of the first press release text to return.
The index of the first text is 1.
If it is not specified, the default value is 1, meaning that the web service will return the texts starting at the first one.
Also, if this number is specified, the texts will be sorted by their date and time and the latest are returned.

Filter

The filter is optional.
If it is not specified, the response will contain all the press release texts.
Please see PressReleaseTextFilter for more information.

Fields

The list of fields is mandatory.
It can contain all the fields in the PressReleaseText data. Only the fields you have access to will be returned.

 Response
When the request is successful, the filter will return the list of filtered press release texts.
 Security

This is a public request: any client can retrieve information.

Some of the fields for a press release text are public and other are not.
The response from the server will contain only the fields the client has access to.

 Example

Retrieve the last 5 press releases texts, for volleyball, in English and that are published on the home page.

Web service request
Copy Code 
<Request Type="GetPressReleaseTextList"
         Number="5"
         Fields="Author DateTime Headline NoMatch PhotoTitle TournamentCode">
  <Filter Category="Volleyball" Language="en" PublishOn="Home" />
</Request>
