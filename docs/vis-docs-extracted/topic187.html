Request to get a registration of a player in a volleyball tournament
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetVolleyPlayer"
         No="<player number>">
         Fields="<Optional: list of the fields to return>" />

Where:

Attribute Description
No The number of the player registration.
This is a mandatory parameter.

Fields

The list of fields is optional: if it not specified, all the fields you have access to will be sent.
It can contain all the fields in the VolleyPlayer data. Only the fields you have access to will be returned.

 Response
When the request is successful, the service will return the data of the registration of a player in a volleyball tournament.
 Errors

In addition to global errors, the following errors can be sent by the server:

Code Description
BadParameter The No parameter is not a valid 32-bits integer number.
NoData The registration with the specified number does not exist or has been deleted.
ParameterMissing You must specify the No parameter.
 Security

This is a public request: any client can retrieve information.

Some of the fields for a registration are public and other are not.
The response from the web service will contain only the fields the client has access to.

 Example

Get the player registration with the number 63812.

Web service request
Copy Code 
<Request Type="GetVolleyPlayer"
         No="63812"
         Fields="No NoShirt IsCaptain IsLibero">
  <Relation Name="Player" Fields="No FirstName LastName FederationCode" />
  <Relation Name="Team" Fields="No Code Name" />
  <Relation Name="Tournament" Fields="No Code Name" />
</Request>
