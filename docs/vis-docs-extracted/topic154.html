Volleyball Team Filter

Filter to select teams registrations in volleyball tournaments.

 Properties

The following table lists the properties for the filter.

Name Type Description
NoTournament Int32 Selects the teams registered in a tournament.
NoTournaments Int32 set Selects the teams registered in multiple tournaments.
 Examples
Filter Description
<Filter NoTournament="979" /> Selects the teams registered in the tournament with the number 979.
