Volleyball Pool

Represents a pool in a volleyball tournament.

The name of the XML element for a player is <VolleyballPool>.
The name of the XML element for a list of players is <VolleyballPools>.

 Properties

The following table lists all the properties for a pool.

Name Type Description
Code String The code.
DeletedDT DateTime Empty if the pool is valid.
Date and time of the deletion when the pool is deleted.
LastChangeDT DateTime The date and time of the last change.
LastChangeUser Int32 The number of the user who has made the last change.
LastChangeUsername String The name of the user who has made the last change.
Location String The location.
Name String The name.
No Int32 The unique number.
NoRound Int32 The number of the round to which the pool belongs.
NoTournament Int32 The number of the tournament to which the pool belongs.
Order Int32

The order.
Can be used to sort the pools in the order of the tournament.

This value is not always incremental, but a value is only used once.
RankingMethod VolleyPoolRankingMethod The method to use to rank the pool.
RoundCode String The code of the round to which the pool belongs.
RoundName String The name of the round to which the pool belongs.
TechnicalInfo VolleyPoolTechnicalInfo The technical information.
Type VolleyPoolType The type.
Version Int32 Version.
The version is global to all pools. It is used when requesting a list of pools, to get only the changes of the list.
