Country flags (small)

You can retrieve the country flags, in a small size (16 x 11 pixels), using the following URL: https://www.fivb.org/Vis2009/Images/Flags/Small/XX.png.
Where XX is the ISO 3166-1 alpha-2 code of the country.

For federations, you need to convert the federation code to the equivalent country code.
If there is no equivalent country (for ENG, NIR, SCO, WAL), you can use the 3 letters federation code to retrieve the flag.

For example, to retrieve the flag for Australia (ISO code AUS), you can use the following URL: https://www.fivb.org/Vis2009/Images/Flags/Small/AU.png.
