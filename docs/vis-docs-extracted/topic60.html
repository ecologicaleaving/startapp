Volleyball Tournament

Represents a volleyball tournament.

The name of the XML element for a tournament is <VolleyballTournament>.
The name of the XML element for a list of tournaments is <VolleyballTournaments>.

 Logos

Due to historical reasons, there are many properties containing the number of the logos: EventLogos, NoImageLogo, NoImageOrganizerLogo, NoImagePublicity and TournamentLogos.

There are 3 logos for a tournament:

  • The logo of the organizer (FIVB, confederation, federation, IOC, ...).
  • The logo of the tournament (in multiple sizes).
  • The logo to use as the publicity.

For client applications that must only display the logos, the best way is to use only the Logos property. This property is calculated from other logos properties and also define the missing tournament logos, if needed.

Otherwise, to get the right logo, you must use the following order:

  1. Check TournamentLogos, to see if the logo is defined.
  2. Check EventLogos, to see if the logo is defined.
  3. Check the individual NoImageXXX properties.

When the number of the logo is found, if it is not 0, you can get the logo with the GetImage.aspx URL.
