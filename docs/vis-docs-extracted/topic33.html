Beach Volleyball Round

Represents a round for a beach volleyball round.

The name of the XML element for a beach volleyball round is <BeachRound>.
The name of the XML element for a list of beach volleyball rounds is <BeachRounds>.

 Fields

The following table lists all the fields for a beach volleyball round.

Name Type Description
Bracket String Bracket.
Code String Code.
If Phase indicates a federation or confederation quota round, this field contains the code of the federation or the confederation.
DeletedDT DateTime Null if the round is valid.
Date and time of the deletion when the round is deleted.
EndDate Date Ending date for the matches in the round.
LastChangeDT DateTime Date and time of the last change in the round's data.
LastChangeUser Int32 Number of the user who makes the last changed of the event.
LastChangeUsername String Username of the user who has made the last change in the round's data.
Name String Name.
NbViewRanking Int32 Statistics: number of views for the ranking.
No Int32 Unique number of the round.
This number is unique among all the registered rounds.
NoInTournament Int32 Number of the round in the tournament.
NoTournament Int32 Unique number of the tournament to which the round belongs.
This field can be used to retrieve the information about the tournament in the BeachTournament data.
Phase BeachRoundPhase Phase.
RankingInformation String Special information about the ranking.
This field should normally contain a text to explain some special cases of the ranking, for example when 2 or more teams are tied.
The text here, if any, can be displayed near the ranking table.
This field may contain formatting elements expressed as HTML codes.
RankMethod BeachRoundRankingMethod Method used for ranking.
StartDate Date Starting date for the matches in the round.
Version Int32 Version of the round's data.
This field is currently not used.
