Beach Volleyball Matches Filter

Filter for a list of beach volleyball matches.

 

Properties

The following table lists the properties for the filter.

Name Type Description
FirstDate Date

First date for the matches.

When this attribute is specified, the filtered matches will contain only the matches played at or after the specified date.
InConfederationQuota String

The code of the confederation for matches that play in a confederation quota selection round.
When this attribute is specified, the filtered matches will contains only those that plays a confederation quota round for the specified confederation.
If you want to retrieve all the confederation quota matches, for all confederations, you can the value of this parameter empty.

Due to the current implementation of the SQL databases, the InConfederationQuota and InFederationQuota filters have the same effect.
It's the code in the attribute that indicates whether the filter is for a confederation or a federation.
InFederationQuota String

The code of the federation for matches that play in a federation quota selection round.
When this attribute is specified, the filtered matches will contains only those that plays a federation quota round for the specified federation.
If you want to retrieve all the federation quota matches, for all confederations, you can the value of this parameter empty.

Due to the current implementation of the SQL databases, the InConfederationQuota and InFederationQuota filters have the same effect.
It's the code in the attribute that indicates whether the filter is for a confederation or a federation.
InMainDraw Boolean

Filtering about the membership of a match in the main draw.

The value of this parameter can be one of the following:

Value Description

Empty
OR
not specified

 The filter does not use the main draw membership.
false The filtered matches will contain only matches that are not in the main draw.
true The filtered matches will contain only matches that are in the main draw.
InQualification Boolean

Filtering about the membership of a match in the qualification tournament.

The value of this parameter can be one of the following:

Value Description

Empty
OR
not specified

 The filter does not use the qualification tournament membership.
false The filtered matches will contain only matches that are not in the qualification tournament.
true The filtered matches will contain only matches that are in the qualification tournament.
LastDate Date

Last date for the matches.

When this attribute is specified, the filtered matches will contain only the matches played at or before the specified date.
NoMatch Int32

Unique number of the match, for head to head information.

When this attribute is specified, the filtered matches will contain only the matches played by the same teams as the match with the specified number.
