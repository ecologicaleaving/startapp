Beach Volleyball Olympic Selection Ranking Entry

Represents an entry in the ranking for a the selection of the beach volleyball teams that can participate in the Olympic Games.
Note that the IOC is working with countries and not federations. That explains why there is a country code field and not a federation code one.

The name of the XML element for an entry for the ranking is <BeachOlympicSelectionRankingEntry>.

 Fields

The following table lists all the fields in the ranking for the selection of the beach volleyball teams that can participate in the Olympic Games.

Name Type Description
GamesYear Int32 Year of the Olympic Games for the ranking.
NbParticipations Int32 Number of participations in the qualifying tournaments.
NoPlayer1 Int32 Unique number of the first player.
This field can be used to retrieve the information about the player in the Player data.
NoPlayer2 Int32 Unique number of the second player.
This field can be used to retrieve the information about the player in the Player data.
Points Int32 Number of points earned in the qualifying tournaments.
Position Int32 Position of the team in the ranking.
Is always from 1 to the number of teams in the ranking.
This field must be used to sort the ranking in the right order.
SelectionRank Int32 Rank of the team if selected.
This field is 0 if the team is not selected.
Status BeachOlympicTeamStatus Status of the team.
TeamCountryCode CountryCode Code of the country for the team.
TeamName String Name of the team.
