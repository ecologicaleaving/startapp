Federation Code

A federation code uses a coding unique to the FIVB, using 3 letters.

A federation is not always the same as a country.

The United Kingdom country contains 4 federations:

  • England.
  • Northern Ireland.
  • Scotland.
  • Wales.
This code should not be confused with IOC codes or ISO-3166-1 alpha-3 country codes.
