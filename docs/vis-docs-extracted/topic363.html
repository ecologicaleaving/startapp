Volley Live Schema
Volley Live Schema
Description

XSD schema that can be used to validate the live data for a volleyball match.

The schema file can be downloaded here.

Elements
ElementDescription
VolleyLiveMain element.

Data for the live information about a volleyball match.

The order of the elements is free.

Depending on the options specified by the caller, the received file can change.

You can download the schema file here.
