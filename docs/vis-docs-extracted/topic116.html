Volley Live Export Option

Option for the export of volleyball live data.

Each option is defined as a value and the values can be combined to a single number. Use the %T:Fivb.Vis.Volley.Live.VolleyLiveExportOptionSet% to specify the values.

 Values

The following table lists the currently available options (in the order of their value):

Value Hex value Name Description

1

0x0001

 Events All the events must be exported.

2

0x0002

 EventsIdentifiers The identifiers of the events must be exported.

4

0x0004

 EventsTimeOffsets The time offsets of the events must be exported.

8

0x0008

 Information Information about the tournament, the match, the referees, the teams and the players must be exported.

16

0x0010

 SetLineUp The line-ups at the start of the sets must be exported.

32

0x0020

 PlayersBallPositions The positions of the players and the ball must be exported.

64

0x0040

 PressComments The press comments must be exported.

128

0x0080

 Score The score must be exported.

256

0x0100

 SetStatistics The statistics of the sets must be exported.

512

0x0200

 RallyLineUp The line-ups of each rally must be exported.

1024

0x0400

 RallyScore The score of each rally must be exported.

2048

0x0800

 MatchStatistics The statistics of the match must be exported.

4096

0x1000

 EventsComments The comments must be exported in the events.

8192

0x2000

 EventsRallies The rallies must be exported in the events.

16384

0x4000

 InformationOnStatistics The statistics must contain information about the player / team.
