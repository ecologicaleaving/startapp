Int32 Set

Set of 32 bits integer numbers.

It is used to specify more than one integer value in a filter.

The set must contain a list of ranges separated by one or more spaces. Each range can be a single item number, or two item numbers separated by a dash. In the latter case, all the items with a number between the limits are selected, including the limits.

Examples

Value Description
1 Only the item whose number is 1.
1-5 The items whose number is between 1 and 5.
1 4-5 The items whose number is 1, 4 or 5.
