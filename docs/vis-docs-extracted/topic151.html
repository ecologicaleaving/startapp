Volleyball Match Filter

Filter to select matches in volleyball tournaments.

 Properties

The following table lists the properties for the filter.

Name Type Description
FirstDate Date Selects the matches at or after the specified date.
ForLiveScore Boolean Selects the matches that are available for live score.
Includes the matches currently in progress and those that will start between the previous 3 hours and the next 30 minutes.
HeadToHeadCodes String Selects the matches by the codes of their teams.
The value must be two team codes separated by a space.
LastDate Date Selects the matches at or before the specified date.
NoEvent Int32 Selects the matches of the specified event.
NoMatches Int32 set Selects the matches with the specified numbers.
NoPool Int32 Selects the matches of the specified pool.
For a play-off pool, the filter selects all the matches that are taken into account for the ranking of the pool. These contains the matches of the pools and the matches of selected previous pools for the qualified teams. To get the selected pools and the qualified teams, you can look at the technical information of the pool.
NoTeam Int32 Selects the matches of the specified team.
Due to a current limitation on the SQL tables, the web service uses the team code for the check. As each team is identified by a unique code, this should not pose problem.
NoTournament Int32 Selects the matches of the specified tournament.
NoTournaments Int32 set Selects the matches of the specified tournaments.
Statuses VolleyMatchStatus set Selects the matches whose status is one of the values.
If this property is specified, it must contains a list of statuses separated with a space. Each status in the list can be specified by its Value or Name.
TournamentGenders EventGender set

Selects the matches whose tournament gender is one of the values.
If this property is specified, it must contains a list of genders separated with a space. Each gender in the list can be specified by its Value or Name.
TournamentOrganizerCode
TournamentOrganizerType		 String
OrganizerType		 Selects the matches whose tournament is organized by the specified one.
You can specify only the organizer type and the selected matches will be those with specified organizer type.
If the organizer type is Confederation or Federation, you can also specify the confederation code or federation code, respectively. In this case, the selected matches will be those organized by the specific confederation or federation.
TournamentTypes VolleyTournamentType set Selects the matches whose tournament type is one of the values.
If this property is specified, it must contains a list of types separated with a space. Each type in the list can be specified by its Value or Name.
 Examples
Filter Description
<Filter NoTournament="979" /> Selects the matches in the tournament with the number 979.

<Filter FirstDate="2018-6-1"
        LastDate="2018-6-30" />

 Selects the matches that are played in June 2018.

<Filter TournamentGender="M"
        TournamentOrganizerType="Fivb" />
        HeadToHeadCodes="BRA USA" />

 Selects the matches between Brazil and USA, for men tournaments organized by the FIVB..

<Filter TournamentOrganizerType="Fivb" />

 Selects the matches in tournaments organized by the FIVB.

<Filter TournamentOrganizerType="Federation"
        TournamentOrganizerCode="USA" />

 Selects the matches in tournaments organized by the USA federation.
