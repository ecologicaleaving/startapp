Tag Item
This documentation is preliminary, as the tags API is currently in development. The information here is subject to changes.

Represents the assignment of a tag to an item.

The name of the XML element for an assignment is <TagItem>.
The name of the XML element for a list of assignments is <TagsItems>.

 Properties

The following table lists all the properties for an assignment.

Name Type Description
ItemType DataType The type of the related item
No Int32 The number.
NoItem Int32 The number of the related item.
NoTag Int32 The number of the tag.
TagNoItem Int32

The number of the referenced item.
For example, if the tag indicates a player, this is the number of the player.

The value can be empty (null in JSON) if there is no referenced item. For example for a system tag.
TagText String The text for the tag, in English.
The value can be used in a filter, to select items with the tag.
TagUserText String

The text for the tag, in user language.
The value cannot be used in a filter, because it can contain a name or a code, instead of the item number.
User language processing is currently not implemented, so this property has the same value as TagText.
Version Int32 The version of the assignment.
