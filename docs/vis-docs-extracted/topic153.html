Volleyball Pool Filter

Filter to select pools in volleyball tournaments.

 Properties

The following table lists the properties for the filter.

Name Type Description
ForRanking Boolean

Ranking / non-ranking pools.

The possible values are:

Value Description
<not specified> The filter selects both ranking and non-ranking pools.
True The filter selects only the ranking pools.
False The filter selects only the non-ranking pools.
NoTournament Int32 Selects the pools of a specified tournament.
Numbers Int32 set Selects the pools with the specified numbers.
 Examples
Filter Description
<Filter NoTournament="979" /> Selects the pools of the tournament with the number 979.

<Filter NoTournament="979"
        ForRanking="True" />

 Selects the ranking pools of the tournament with the number 979.
