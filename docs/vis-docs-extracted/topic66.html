Beach Match Status

Status for a beach volleyball match.

 Values

The following table lists the values for the statuses for a volleyball match.

Value

 Name Description

1

 Scheduled Match scheduled.

2

 ReadyToStart Match is ready to start.

3

 InSet1 Set 1 is running.

4

 Set1Finished Set 1 is finished.

5

 InSet2 Set 2 is running.

6

 Set2Finished Set 2 is finished.

7

 InSet3 Set 3 is running.

8

 Set3Finished Set 3 is finished.

9

 InSet4 Set 4 is running.

10

 Set4Finished Set 4 is finished.

11

 InSet5 Set 5 is running.

12

 Finished Match is finished.

13

 OfficialResult The match result is official.

14

 Corrected The match result has been corrected.

15

 Closed Closed: nothing more can be changed.
 Remarks

There is no status to indicate that the set 5 is finished, because a match can have at most five sets and, if there is a fifth set, the end of set 5 is the end of the match.

If the match is played in only one set, the status following InSet1 is Finished.
If the match is played in only three sets, the status following InSet3 is Finished.

If a match is played in three sets and a team wins the first two sets, the status following InSet2 is Finished.
If a match is played in five sets and a team wins after 3 sets, the status following InSet3 is Finished. If a team wins after 4 sets, the status following InSet4 is Finished.
