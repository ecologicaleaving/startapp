Tags Filtering

You can use tags to filter by article.

When tags are specified in a filter, they are in a child XML of the main filter tag, named <Tags>. In this element, you can specify one or more tags, separated by a space or an end of line. If more than one tag child is specified, the filter selects the item that have all the specified tags.

Here are some examples (note that different separator styles are used for the tags):

The news for volleyball tournament 979 (WL2017-1)
Copy Code 
<Tags>#new volley-tournament:979</Tags>
The profiles for player 110419
Copy Code 
<Tags>
  #player-profile
  player:110419
</Tags>
The articles about volleyball match 226039
Copy Code 
<Tags>volley-match:226039</Tags>
The articles referenced by my-own-tag
Copy Code 
<Tags>
  my-own-tag
</Tags>
