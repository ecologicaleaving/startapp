Volleyball Team

Represents the registration of a team in a volleyball tournament.

The name of the XML element for a team is <VolleyballTeam>.
The name of the XML element for a list of teams is <VolleyballTeams>.

This page contains the following information:

  • Officials.

 Officials

Functions

There are 7 officials with pre-defined functions and 5 with free-function.

Names

For each official, there is 3 properties for the name of an official: xxxFirstName, xxxLastName and xxxName.

Initially, only the name was available but, during the 2018 season, the first name and last name were added. For old team registrations, the last name property contains the full name of the official and the first name is empty. Getting the name returns always the full name, so the applications that don't know the new properties still work as expected.

For new or updated applications, if you only need to display the name of the official, you only need the xxxName property. If you want to separate the first name and last name of the officials, you need the xxxFirstName and xxxLastName properties.

For applications that set the name of the officials, you must use the xxxFirstName and xxxLastName properties, because the xxxName property can't be set.
