Request to get a volleyball match
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetVolleyMatch"
         No="<match number>">
         Fields="<Optional: list of the fields to return>" />

Where:

Attribute Description
No The number of the match.
This is a mandatory parameter.

Fields

The list of fields is optional: if it not specified, all the fields you have access to will be sent.
It can contain all the fields in the VolleyMatch data. Only the fields you have access to will be returned.

 Response
When the request is successful, the service will return the data of a volleyball match.
 Errors

In addition to global errors, the following errors can be sent by the server:

Code Description
BadParameter The No parameter is not a valid 32-bits integer number.
NoData The match with the specified number does not exist or has been deleted.
ParameterMissing You must specify the No parameter.
 Security

This is a public request: any client can retrieve information.

Some of the fields for a match are public and other are not.
The response from the web service will contain only the fields the client has access to.

 Example

Get the match with the number 7604.

Web service request
Copy Code 
<Request Type="GetVolleyMatch" No="7604" Fields="DateTimeLocal DateTimeUtc City Hall TeamAShirtColor TeamBShirtColor MatchPointsA MatchPointsB EndTime NbSpectators">
  <Relation Name="Pool" Fields="Code Name Location Type" />
  <Relation Name="TeamA" Fields="Code Name" />
  <Relation Name="TeamB" Fields="Code Name" />
  <Relation Name="Tournament" Fields="Code Name" />
</Request>
