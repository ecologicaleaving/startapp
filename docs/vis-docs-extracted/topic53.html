Volleyball Player

Represents the registration of a player in a volleyball tournament.

The name of the XML element for a player is <VolleyballPlayer>.
The name of the XML element for a list of players is <VolleyballPlayers>.

This page contains the following information:

  • Players selection

 Players selection

The club, country or federation first selects a maximum of MaxNbPlayersO2A players. The delay for this selection is given by DeadlineO2A. Not all tournaments have this steps, some of them skip this step to go directly to the following one.

Then, a maximum of MaxNbPlayersO2 players must be pre-selected and the players are given a shirt number, The delay for the preselection is given by DeadlineO2.

Finally, a maximum of MaxNbPlayersO2bis must be selected to play in the tournament. The delay for the selection is given by Deadline O2bis.

All the properties (the maximum number of players and the deadlines, are defined in the tournament.
