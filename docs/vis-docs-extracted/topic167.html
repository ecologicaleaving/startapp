Request to get a beach volleyball team
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetBeachTeam"
         No="<team number>"
         Fields="<optional list of the fields to return" />

Where:

Attribute Description
No Number of the beach volleyball team.
This is a mandatory parameter.

Fields

The list of fields is optional: if it not specified, all the fields you have access to will be sent.
It can contain all the fields in the BeachTeam data. Only the fields you have access to will be returned.

 Response
When the request is successful, the service will return the data of a beach volleyball team.
 Errors

In addition to global errors, the following errors can be returned by the server:

Error Description
<BadParameter> The No parameter is not a valid 32-bits integer number.
<NoData> The beach volleyball team with the specified number does not exist or has been deleted.
<ParameterMissing> You must specify the No parameter.
 Security

This is a public request: any client can retrieve information.

Some of the fields for a beach volleyball team are public and other are not.
The response from the server will contain only the fields the client has access to.

 Example

Retrieve the team with the number 375442.

Web service request
Copy Code 
<Request Type="GetBeachTeam"
         No="375442"
         Fields="NoPlayer1 NoPlayer2 Name Rank EarnedPointsTeam" />

The response is (formatted for readability):

Web service response
Copy Code 
<BeachTeam NoPlayer1="118573"
           NoPlayer2="102966"
           Name="Candelas-Garcia"
           Rank="13"
           EarnedPointsTeam="180"/>
