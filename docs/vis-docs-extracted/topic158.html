Volleyball competitions

Tournament

The entry point to display information about volleyball competitions is the volleyball tournament data type. Knowing the number of a tournament, you can get information about its teams, players, rounds, matches, rankings, ...

To get the list of the tournaments, you must use the GetVolleyTournamentList request.
You must specify the properties that the web service should return with the Fields attribute.
By default the web service returns all the volleyball tournaments. To limit the returned tournaments, you can use the VolleyTournament filter.

To get information about a tournament, you must use the GetVolleyTournament request.
With this request, the web service returns all the properties you have access to. If you want to limit the returned properties, you can use the Fields attribute.

To get the ranking of a tournament, you must use the GetVolleyTournamentRanking request.

To get the ranking of the players of a tournament, you must use the GetVolleyPlayersRankingList request.

Players

You can get the players registrations with the GetVolleyPlayerList request.
You must specify the properties that the web service should return with the Fields attribute.
By default the web service returns all the players registrations. To limit the returned registrations (for example the registrations of a tournament or of a team), you can use the VolleyPlayer filter.

To get information about a player registration , you must use the GetVolleyPlayer request.
With this request, the web service returns all the properties you have access to. If you want to limit the returned properties, you can use the Fields attribute.

GetVolleyPlayer and GetVolleyPlayerList return information about the registration of players in tournaments and there can be more than one entry for a specified player, in different tournaments, if he/she is registered in more than one tournament. To get information about the player himself/herself, you can use the GetPlayer request, that returns global information about the player.

Live information

If you want to display a match while it is played, you must use the GetVolleyLive request.
This request takes an Options parameter that indicates what kind of data you want.

To limit the used bandwidth, this request uses the Version parameter. For the first request, you don't specify this parameter and the web service returns the requested live information. For the following request, you specify the version of the live information you already have and the web service returns with a NoChanges status code to indicate that your data is up-to-date.
