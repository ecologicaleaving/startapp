Volley Pool Ranking Method
Method of ranking for a volleyball pool.
 Values

The following table lists the values for the ranking methods for a volleyball pool.

The match points are attributed as follows:

Value

 Name Description

1

 None None.
This ranking method is used when there is no algorithm to automatically rank the pool.
This kind of pool generally defines the ranks using the LoserRank and WinnerRank fields from the match data.

2

 2010-1

For matches played in 5 sets.

The match points are attributed as follows:

Result Points
Won by 3-0 or 3-1

3
Won by 3-2

2
Lost by 2-3

1
Lost by 1-3 or 0-3

0

The ranking is then made using the following comparisons:
  1. Number of match points.
  2. Number of matches won.
  3. Points ratio.
  4. Sets ratio.

In case of tie, the teams will be displayed by their alphabetic team code.
