VIS web service documentation
Data Types

The following table lists all the data type managed by the VIS web service.

Name Description
Article Article.
ArticleText Text for an article.
BeachMatch Match in a beach volleyball tournament.
BeachOlympicGamesSelectionRankingEntry Entry for the ranking for the selection of the teams for Olympic Games, for beach volleyball.
BeachRound Round in a beach volleyball tournament.
BeachRoundRankingEntry Entry for the ranking of a round in a beach volleyball tournament.
BeachTeam Team in a beach volleyball tournament.
BeachTechnicalPlayersRankingEntry Entry for the players Technical ranking, for beach volleyball.
BeachTechnicalTeamsRankingEntry Entry for the teams Technical ranking, for beach volleyball.
BeachTournament Beach volleyball tournament.
BeachTournamentRankingEntry Entry for the ranking of a beach volleyball tournament.
BeachWorldTourRankingEntry Entry for the World Tour ranking for beach volleyball.
Event Event.
EventLogos Logos for an event.
Player Player.
PoolRankingRankList Ranking of the teams eliminated in a pool or a round.
PressRelease Press release.
PressReleaseText Text for a press release.
Tag Tag.
TagItem Assignment of a tag to an item.
TagText Text for a tag.
VolleyMatch Volleyball match.
VolleyPlayer Registration of a player in a volleyball tournament.
VolleyPlayersRanking Ranking of players in a volleyball tournament.
VolleyPool Volleyball pool.
VolleyPoolRankingEntry Entry in the ranking of a volleyball pool.
VolleyPoolTechnicalInfo Technical information about a volleyball pool.
VolleyTeam Registration of a team in a volleyball tournament.
VolleyTournament Volleyball tournament.
VolleyTournamentRankingEntry Entry in the ranking of a volleyball tournament.
VolleyTournamentRankingTeamSource Source of the team for the ranking of a volleyball tournament.
