Request to get a list of beach volleyball rounds
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetBeachRoundList
         Fields="<list of the fields to return>">
  <Filter /> <!-- Optional: contains the filter to use -->
</Request>

Filter

The filter is optional.
If it is not specified, the response will contain all the beach volleyball rounds.
Please see the Beach Volleyball Round Filter description for more information.

Fields

The list of fields is mandatory.
It can contain all the fields in the Beach Volleyball Round data. Only the fields you have access to will be returned.

 Response
When the request is successful, the filter will return the list of filtered beach volleyball rounds.
 Security

This is a public request: any client can retrieve information.

Some of the fields for a beach volleyball round are public and other are not.
The response from the server will contain only the fields the client has access to.

 Example

Get the list of rounds for tournament 502.

Web service request
Copy Code 
<Request Type="GetBeachRoundList"
         Fields="Code Name Bracket Phase StartDate EndDate" />
  <Filter NoTournament="502"/>
</Request>
