Request to get a beach volleyball World Tour ranking
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetBeachWorldTourRanking"
         Gender="<gender>"
         Number="<number of entries to return>"
         ReferenceDate="<ranking reference data>
         Fields="list of the fields to return" />

Where:

Attribute Type Description
Gender Gender Gender of the teams to retrieve the ranking for.
This is a mandatory parameter.
Number Int32 Number of entries to return.
If this parameter is not specified, all the entries in the ranking will be returned.
ReferenceDate Date Reference date of the ranking: when the ranking has been calculated.
If there is no World Tour ranking calculated at the specified date, the returned ranking will be empty.
If this parameter is not specified, the web service will return the ranking for the latest reference date.
 Response
When the request is successful, the service will return the a <BeachWorldTourRanking> XML element that contains beach World Tour ranking entries.
 Errors

In addition to global errors, the following errors can be sent by the server:

Error Description
<BadParameter>
  • The Gender parameter has not a valid value.
  • The Number parameter, if specified, is not a valid 32-bits integer number.
  • The ReferenceDate parameter, if specified, is not not a valid date.
<ParameterMissing> You must specify the Gender parameter.
 Security

This is a public request: any client can retrieve information.

All the fields for a beach volleyball World Tour ranking are public.

 Example

Retrieve the latest World Tour ranking for women.

Web service request
Copy Code 
<Request Type="GetBeachWorldTourRanking"
         Gender="W"
         Number="10"
         Fields="Position Rank TeamName TeamFederationCode NbParticipations EarnedPoints EarningsTotalTeam" />
