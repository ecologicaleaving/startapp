CEV Beach Federation Ranking Method

Ranking method used for CEV beach volleyball federation ranking.

 Values

The following table lists the values for the ranking methods.

Value Name Description
1 BestPlayersSum_EntryPoints The number of points for a federation is the sum of the entry points of the NbPlayers best players of the federation.
