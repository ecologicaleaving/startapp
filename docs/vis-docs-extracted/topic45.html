Article
This documentation is preliminary, as the articles API is currently in development. The information here is subject to changes.

Represents an article.
This can be press releases, person profiles, information about a tournament or an event.
An article contains only data that is not tied to a language. Each article can contain one or more texts, available with the Texts relation.

This data type supersedes the PressRelease data type.

The name of the XML element for an article is <Article>.
The name of the XML element for a list of articles is <Articles>.
