Request to get the ranking of a volleyball tournament
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetVolleyTournamentRanking"
         No="<tournament number>"
         Fields="<list of the fields to return>" />

Where:

Attribute Description
No The number of the volleyball tournament.
This is a mandatory parameter.

Fields

The list of fields is mandatory.
It can contain all the fields in the VolleyTournamentRankingEntry data. All the fields are public, so you can request any one.

 Response
When the request is successful, the service will return the a <VolleyballTournamentRanking> XML element that contains volleyball tournament ranking entries.
 Errors

In addition to global errors, the following errors can be sent by the server:

Code Description
BadParameter The No parameter is not a valid 32-bits integer number.
NoData The volleyball tournament with the specified number does not exist or has been deleted.
ParameterMissing You must specify the No parameter.
 Security

This is a public request: any client can retrieve information.

All the fields for a volleyball tournament ranking are public.

 Example

Get the ranking for the tournament with the number 1032.

Web service request
Copy Code 
<Request Type="GetVolleyTournamentRanking"
         NoTournament="1032"
         Fields="Rank TeamSource TeamCode TeamName" />
