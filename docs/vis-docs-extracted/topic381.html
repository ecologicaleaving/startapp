Rally
Rally.
Duration optional xs:nonNegativeInteger
The duration, in ms. The attribute is optional: it is omitted if the duration can't be known. For this value to be calculated, the start time offset and the end time offset must be known.
EndTimeOffset optional TimeOffset Simple Type
Time offset of the end of the rally, from the start of the match. The attribute is omitted if the rally is terminated or the time offset is unknown.
Identifier optional UUID Simple Type
Identifier of the event. This is a UUID.
IsKey optional xs:boolean
Indicates whether the rallyis a key rally If the attribute is missing, the rally is not a key rally.
IsMatchPointTeamA optional xs:boolean
Indicates that team A will win the match if it wins this rally. If the attribute is missing, the rally is not for a match point. Otherwise, the IsSetPointTeamA attribute gives the ordinal number of match points for the set.
IsMatchPointTeamB optional xs:boolean
Indicates that team B will win the match if it wins this rally. If the attribute is missing, the rally is not for a match point. Otherwise, the IsSetPointTeamB attribute gives the ordinal number of match points for the set.
IsSetPointTeamA optional xs:nonNegativeInteger
Indicates that team A will win the set if it wins this rally. If the attribute is missing, the rally is not for a set point. The value of the attribute is the ordinal number of set points for the team in the set.
IsSetPointTeamB optional xs:nonNegativeInteger
Indicates that team B will win the set if it wins this rally. If the attribute is missing, the rally is not for a set point. The value of the attribute is the ordinal number of set points for the team in the set.
No optional xs:positiveInteger
Number of the rally. The rallies are numbered starting at 1 for the first rally of the match. The attribute is omitted if the rally numbers are not requested from the web service.
PointsTeamA optional xs:nonNegativeInteger
The number of points for team A at the end of the rally. The attribute is omitted if the rally is not finished.
PointsTeamB optional xs:nonNegativeInteger
The number of points for team B at the end of the rally. The attribute is omitted if the rally is not finished.
Status optional Restriction of VolleyRallyStatus Simple Type
Status. If the attribute is missing, the rally is accepted.
TimeOffset optional TimeOffset Simple Type
Time offset of the rally, from the start of the match.
UserSetsScore optional xs:boolean
Whether the score has been set by the user. If the attribute is missing, the score is calculated automatically.
