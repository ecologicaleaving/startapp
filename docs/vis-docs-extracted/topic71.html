Discrete values > Beach volleyball > Beach Team Type
Beach Team Type

Type for a team in a beach volleyball tournament.

 Values

The following table lists the values for the types for a team in a beach volleyball tournament.

Value

 Name Description

0

 Normal Normal.

1

 WildCard The team has received a wild card.

2

 TechnicalWildCard The team has received a technical wild card.

3

 LateEntry This is a late entry team.

4

 Reserve This is a reserve team.

5

 HostTeam This is a team from the host country.

6

 QualificationWildCard The team has received a qualification tournament wild card.

7

 Confederation The team is selected by the confederation.

8

 FederationSpot This is not a real team, but an indication that this spot is reserved for a team from a federation selection round.

9

 ConfederationSpot This is not a real team, but an indication that this spot is reserved for a team from a confederation selection round.

10

 OrganizerVacancy The team has been selected because its federation organizes a tournament in the World Tour.
