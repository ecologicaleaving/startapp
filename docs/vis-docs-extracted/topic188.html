Request to get a list of registration of players in volleyball tournaments
 Request

The syntax for this request is:

Web service request
Copy Code 
<Request Type="GetVolleyPlayerList"
         Fields="<list of the fields to return>">
         Version="<version of local list>">
  <Filter /> <!-- optional: contains the filter to use -->
</Request>

Fields

The list of fields is mandatory.
It can contain all the fields in the VolleyPlayer data. Only the fields you have access to will be returned.

Filter

The filter is optional.
If it is not specified, the response will contain all the players registrations.
Please see the VolleyPlayerFilter description for more information.

Version

The version is optional.
The goal of this parameter is to request an update to an existing local list: for the first request, you don't specify the parameter (or set it to 0) and you receive the full list of the filtered registrations; for subsequent requests, you specify the version of the local list and the web service only returns the changes (new registrations or registrations that have changed). Doing this will decrease the bandwidth used by the local application.
Note that the version of all the registrations (not only the filtered ones) is returned in the root of the response.

 Response

When the request is successful, the web service returns the filtered list of players registrations.

In addition, the root element contains the number of registrations that are returned and the version of all the registrations.

 Security

This is a public request: any client can retrieve information.

Some of the fields for a player registration are public and other are not.
The response from the web service will contain only the fields the client has access to.

 Example

Get the list of the players registered in the team registration with the number 3004.

Web service request
Copy Code 
<Request Type="GetVolleyPlayerList"
          Fields="No">
  <Filter NoTeam="3004" />
  <Relation Name="Player" Fields="No FirstName LastName" />
</Request>
