Volleyball > Competitions > Pools > Volley Pool Type
Volley Pool Type

Type for a volleyball pool.

 Values

The following table lists the values for the types.

Value

 Name Description Ranking

1

 Ranking Ranking.
The teams are ranked from the results of the matches in the pool.



2

 DirectElimination Direct elimination.
The teams are eliminated when they lose one or more matches.

3

 PlayOff Play-off.
The teams are ranked from the results of the matches in this pool and some other pools.



4

 Classification Classification.
The teams are ranked from the LoserRank and WinnerRank fields from the match data.


The Ranking column indicates whether this pool type has a ranking.
