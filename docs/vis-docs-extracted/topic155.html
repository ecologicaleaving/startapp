Volleyball Tournament Filter

Filter to select volleyball tournaments.

 Properties

The following table lists the properties for the filter.

Name Type Description
CountryCode CountryCode Selects the tournaments that are played in the specified country.
FirstDate Date Selects the tournaments that end at or after the specified date.
Genders EventGender set Selects the tournaments whose team gender is one of the value.
If this property is specified, it must contains a list of event genders separated with a space. Each gender in the list can be specified by its Value or Name.
IsVisManaged Boolean

Indicates how the filter selects the tournaments based on the fact that they are managed by the VIS system.
The property can take the following values:

Value Description
not specified or empty The filter doesn't check whether the tournaments are managed by the VIS system or not.
True The filter selects the tournaments that are managed by the VIS system.
False The filter selects the tournaments that are not managed by the VIS system.
LastDate Date Selects the tournaments that end at or before the specified date.
NoEvent Int32 Selects the tournaments of the specified event.
Numbers Int32 set Selects the tournaments with the specified numbers.
PublishOnMsdp Boolean

Indicates how the filter selects the tournaments based on the fact that they must be published on the Microsoft Sports Digital Portal.
The property can take the following values:

Value Description
not specified or empty The filter doesn't check whether the tournaments must be published on the MSDP or not.
True The filter selects the tournaments that must be published on the MSDP.
False The filter selects the tournaments that must not be published on the MSDP.
Seasons String set Selects the tournaments in the specified seasons.
Statuses VolleyTournamentStatus set Selects the tournaments whose status is one of the values.
If this property is specified, it must contains a list of statuses separated with a space. Each status in the list can be specified by its Value or Name.
 Examples
Filter Description
<Filter Genders="M"
        Season="2017" />		 Selects the tournaments for men that are defined in the 2017 season.

<Filter Genders="M W"
        Season="2017"
        Statuses="Confirmed" />

 Selects the tournaments for men and women that are confirmed in the 2017 season.

<Filter FirstDate="2018-6-1"
        LastDate="2018-7-31"
        Statuses="Confirmed" />

 Selects the tournaments that are confirmed and that end during June and July 2018.
