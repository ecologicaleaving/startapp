GetImage.asmx

Because images can be displayed by a standard HTML page, the retrieving of an image data is made using a special URL: https://www.fivb.org/Vis2009/Images/GetImage.asmx.

Most of the images on the server are JPEG images, but there can also be PNG and WMF images. The images computed by the web service are always JPEG images.

 Parameters

The following table lists the parameters of the image that are specified in the URL as a query string.

Parameter Description
Height Height of the image, in pixels.
MaxSize Maximum size of the image, in pixels.
Setting this parameter is the same as setting the Height and Width parameter with the same value.
No

Number of the image.
The meaning of the this parameter depends on the Type parameter:

Type Meaning
  Number of the image, as returned by the GetImageList request.
Player Number of the player.
Press Number of the press release (not the number of the press release text).
Stretch

Indicates how the image must be stretched.

Value Meaning
None The image must not be stretched.
Fill The image is resized to fill the specified dimensions without preserving the aspect ratio.
Uniform The image is resized to fit in the destination dimensions while it preserves its native aspect ratio.
If the aspect ratio of the destination rectangle differs from the source, the thumbnail image will be smaller than the destination rectangle.
This is the default value.
UniformToFill The image is resized to fill the destination dimensions while it preserves its native aspect ratio.
If the aspect ratio of the destination rectangle differs from the source, the source content is clipped to fit in the destination dimensions.

This parameter is only available for a thumbnail.
Type

Type of the image to retrieve:

Value Description
  Normal image.
Player Image for a player.
Press Image for a press release.
Width Width of the image, in pixels.

If no image correspond to the specified parameters, an empty image of 1x1 pixel will be sent to the client.

If no image size is specified, the returned image will be the original image, as saved on the server.
If an image size is specified, the server will compute and return an image of the specified size.
If the image size is greater than the original image size, the original image will be returned.
In any case, the image will keep its original aspect ratio, excepted if the stretch option is set to Fill.

 Examples

Here is the URL to retrieve the image 200600001, with a maximum size of 150 pixels: https://www.fivb.org/Vis2009/Images/GetImage.asmx?No=200600001&MaxSize=150.

Here is the URL to retrieve the image for the player 135198, with a maximum size of 150 pixels: https://www.fivb.org/Vis2009/Images/GetImage.asmx?Type=Player&No=135198&MaxSize=150.

Here is the URL to retrieve the image for the press release 27808, with a maximum size of 150 pixels: https://www.fivb.org/Vis2009/Images/GetImage.asmx?Type=Press&No=27808&MaxSize=150.

You can click on any URL to see the result of the request.
