VolleyLiveUpload Schema
VolleyLiveUpload Schema
Description

Data to upload for the live information about a volleyball match.

The order of the elements is free.

Depending on the source computer, only some of the elements can be uploaded.

You can download the schema file here.

Elements
ElementDescription
VolleyLiveMain element with all the live data.
Complex Types
Complex TypeDescription
ChallengeCommon data for a challenge.
CommentComment.
InjuryInjury.

PositionPosition of a player or the ball.

TeamSanctionSanction for a team.
TeamStatisticsStatistics for a team.
VolleyLiveLineUpLine-up on the court.
VolleySetLineUpLine-up for a set.
Simple Types
Simple TypeDescription
CommentCategoryCategory for a comment.
SanctionTypeType of sanction.
TimeOffsetTime offset, in ms. If the value is positive, it is relative to the start of the match. If the value is negative, it is relative to the scheduled match date and time.
UUIDUUID, used to uniquely identifies an item.
VolleyMatchResultTypeType of result.
VolleyMatchStatusStatus of the match.
VolleyNoteNote for a skill.
